Data & API — Live Connections
Optional JSON endpoints for terminals and product flows (must return JSON arrays, CORS-enabled). For live vessel tracking, use the AISStream.io card below instead.
🛰 Live AIS — AISStream.io (WebSocket)
Free real-time vessel positions streamed directly to the browser. Get a free key at aisstream.io (no card required). The stream subscribes to areas around your terminals and shows live tankers on the Ship Tracking page.
Note: the key is used from the browser (AISStream keys are free & read-only). AIS broadcasts don't include DWT or cargo, so those columns show "—". Coverage is terrestrial (near-coast), strongest around ports.
