Data & API — Live Connections
Optional JSON endpoints for terminals and product flows (must return JSON arrays, CORS-enabled). For live vessel tracking, see the VesselAPI card below.
🚢 Live AIS — VesselAPI.com (recommended)
Real vessel positions via REST, proxied through a Netlify Function so the key never reaches the browser. Terrestrial AIS coverage (coastal & port areas) — matches our terminal use case.
  1. Sign up free at vesselapi.com (no card) → Dashboard → API Keys → copy your key.
  2. In Netlify → Site configuration → Environment variables, add:
    VESSELAPI_KEY = your key, then trigger a redeploy.
  3. Come back here and click Test — or just open Ship Tracking and click "Refresh Vessels".
⚙ Quota-safe by design: the free plan allows ~150 calls/month. The function keeps a shared cache (Firestore) and refreshes once a day: 4 terminals for positions (rotating through your top 20, full cycle every 5 days) + every other day, 1 vessel gets enriched with its type, flag, DWT, destination & ETA (2 calls each). Capped hard at 145 calls/month regardless of traffic. Enrichment (type/flag/DWT/destination) starts blank and fills in gradually — each vessel is looked up once and cached ~2 weeks, since a ship's identity and destination rarely change mid-voyage. Live GPS position, speed, course & nav-status come with every position refresh; cargo isn't published via AIS so it stays "—".
💹 Market Prices — OilPriceAPI (commodities) + Frankfurter (FX/metals)
Powers the Prices tab: WTI, Brent, Gasoline, Natural Gas (via OilPriceAPI — real-time spot prices), plus currencies and precious metals (via Frankfurter — free, keyless, no setup needed for that part).
  1. Register free (no card) at oilpriceapi.com → copy your API token.
  2. In Netlify → Site configuration → Environment variables, add:
    OILPRICEAPI_KEY = your token, then trigger a redeploy.
  3. Come back here and click Test, or open the Prices tab directly.
⚙ Quota-safe: OilPriceAPI's free tier is 200 requests/month. All 4 commodities are fetched in ONE call, cached in Firestore, and refreshed at most twice a day (~60 calls/month, hard-capped at 180). The free tier returns current prices only, so trend charts build up over time — each daily refresh appends one real data point. Currencies/metals via Frankfurter need no key and work immediately.
